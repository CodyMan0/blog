# 영어 트랙

## 목표

**2028년 1월까지 중상급 회화 실력 (B2~C1 수준).**

영어영문학과 출신이지만 오랜 시간 안 써서 굳어있는 영어를 다시 *"쓰는 영어"*로 되살리는 것이 핵심입니다.
문법·독해 기반이 어느 정도 있다는 전제 위에 **회화 / 발화**를 집중 강화합니다.

## 페이즈 (총 ~20개월)

| 페이즈 | 기간 | 내용 |
|---|---|---|
| **Phase 1** | 2026.05 ~ 08 (M1~3) | 정철대학 엔진 1·2·3 — 기본기 인풋 재정비 |
| **Phase 2** | 2026.09 ~ 2027.01 (M4~8) | 미드 쉐도잉 — 발화·청취 동시 훈련 |
| **Phase 3** | 2027.02 ~ 2028.01 (M9~20) | 1:1 회화 강화 + 외국계 인터뷰 영어 |

### Phase 1 디테일

머리 가장 맑은 출근 전 30분 + 자투리 점심 15분.
영어영문학과 시절에 한 번 다진 문법을 의식적으로 다시 깨우기.

### Phase 2 디테일

친숙한 시리즈 1편씩 골라 chunk 단위로 따라 말하기.
*추천: 일·생활 영역 시리즈 (Friends, Modern Family, The Office, Suits 등)*

### Phase 3 디테일

- 1:1 대화 빈도/난이도 단계적 상승
- 외국계 면접 준비 (behavioral STAR, 코드 설명 thought process, system design 어휘)
- Pramp / interviewing.io 모의 면접

## 선생님 수업 (병행)

화·목 1:1 수업 — 그 주에 마주친 약점/표현/면접 질문을 가져가서 다듬기.
**주간 글쓰기 메타 회고를 첨삭 자료로 활용** → 글쓰기 트랙과 자동 연계.

## 일일 루틴

| 시간 | 활동 |
|---|---|
| 출근 전 30min | 인풋 (정철엔진 → 미드 쉐도잉) |
| 점심 15min | 듣기 (팟캐스트, 짧은 영상) |
| 퇴근 후 15min | 코테 풀이를 **영어로** 적기 (output 훈련) |

**일일 미니멈**: 한 표현이라도 입으로 뱉기. 안 한 날은 공식 휴식일.

## 주간 회고

좌측 사이드바의 **주간 회고**에서 매주 표현·실수·다음 주 목표를 정리합니다.
