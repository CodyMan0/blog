# 글쓰기 트랙

## 목표

**사내 PRD 작성을 통해 구조적 사고 + 논리적 글쓰기 훈련.**

PRD는 모호한 아이디어를 *왜 / 무엇을 / 어떻게 / 어디까지*로 분해하도록 강제하는 장치.
이걸 매주 쓴다는 것 자체가 사고 훈련입니다.

## 왜 PRD가 좋은 글쓰기 훈련인가

- **독자가 명확**: 동료/리더가 읽고 의사결정해야 함 → 모호하면 즉시 피드백이 옴
- **구조 강제**: Problem → Goal → Non-goal → Solution → Risk 순으로 사고가 정렬됨
- **결론 우선**: TL;DR 없으면 안 읽힘 → Minto Pyramid의 실전 도장
- **즉각 검증**: 회의에서 받는 질문이 곧 글의 빈틈을 드러냄

## 주차별 운영

매주 사내에서 **PRD 1편 (또는 큰 PRD의 1 섹션)** 작성 후, 토요일에 **메타 회고** 작성:

- 이번 PRD에서 막혔던 문장/구조 1개 (익명화)
- 받은 피드백 → 깨달은 글쓰기 원칙 1개
- 다음 주에 적용할 한 가지

PRD 원문 자체는 사내 자산이라 공개 X. 회고는 *글쓰기 패턴*만 추출.

## 작성 원칙 (장기 운영)

### 1. 발행 우선, 완벽 X
70% 품질로 매주 vs 100% 품질로 한 달에 1번 — 전자가 100배 낫습니다.

### 2. 매주 학습 1개
글쓰기 책 / 좋은 글 / 받은 피드백에서 **그 주의 학습 1개**만 추출.
1년 모이면 *내 글쓰기 변화 곡선*이 보입니다.

### 3. 영어와 결합 (옵션)
가끔 PRD 한 섹션을 영어로 써보기 — 영어 트랙과 시너지.

## 참고 자료 (점진적으로 채우기)

- [Minto Pyramid Principle](https://en.wikipedia.org/wiki/Pyramid_principle) — 결론 먼저, MECE
- 책: 『유시민의 글쓰기 특강』, 『On Writing Well』, 『기술의 표현』
- 첨삭 루트: 영어 선생님 (영문 PRD 섹션) / 동료·리더 (PRD 본문)

## 주간 회고

좌측 사이드바의 **주간 회고**에서 매주 PRD 메타 회고를 정리합니다.
