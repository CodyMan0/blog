# LeetCode 트랙

## 목표

외국계 / 빅테크 코딩 인터뷰 대비. 단순히 문제를 푸는 것을 넘어,
**영어로 사고 과정을 설명하는 훈련**을 함께 진행합니다.

## 페이스 (부담없게)

- **평일**: 1문제 × 30분 캡 — 못 풀면 해설 보고 다음 날 다시 풀기
- **주말**: 한 주 푼 문제 복습 + 주간 회고 작성
- **교재**: [NeetCode 150](https://neetcode.io) 로드맵 따라가기

## 12개월 로드맵

| 기간 | 패턴 | 누적 문제 수 |
|---|---|---|
| M1~2 | Arrays & Hashing, Two Pointers, Sliding Window | ~25 |
| M3~4 | Stack, Binary Search, Linked List | ~50 |
| M5~6 | Trees, Tries, Heap | ~80 |
| M7~9 | Backtracking, Graphs | ~110 |
| M10~12 | DP (1D/2D), Greedy, Intervals | ~150 |

이후 Y2부터는 Hard 위주 + Mock Interview (Pramp, interviewing.io)로 전환합니다.

## 주간 회고 작성 원칙

- **풀이 설명 (approach)**: 영어 — 면접 thought process 훈련
- **회고 (reflection)**: 한국어 — 인지 부담 줄이고 솔직하게
- **그 주 배운 영어 표현**을 자연스럽게 녹여낼 것 (억지로 X)
- 점진적으로 영어 비중 확대: Q1 30% → Q2 50% → Q3 70%

## 진행 현황

좌측 사이드바의 **주간 회고** 목록에서 확인할 수 있습니다.
