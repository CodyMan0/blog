# 외주 대체 프로토타입 구현

> **성과 한 줄 요약**: 외주 750만원 작업을 직접 구현, 2개월간 32개 기능 배포 + 구솔루션 완전 교체

## 상황 (Situation)

신규 솔루션 개발을 위해 외주(750만원)가 검토되고 있었습니다.

## 과제 (Task)

외주 없이 직접 프로토타입을 구현하여 비용을 절감하고, 빠르게 구솔루션을 교체하는 것이 목표였습니다.

## 행동 (Action)

_구체적인 기술적 행동을 추후 작성합니다._

- 랜딩, 백엔드, 솔루션, 어드민 전체 프로토타입 직접 구현
- Claude Code 도입으로 개발 속도 2배 향상

## 결과 (Result)

- 외주 비용 **750만원 절감**
- 2개월간 **32개 기능 배포**
- 구솔루션 **완전 교체 완료**

## 사용 기술

Next.js, React, Claude Code
