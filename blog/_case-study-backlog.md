# 🎯 이직용 케이스 스터디 백로그

> 회고(기록)에서 "증거가 되는 사건"만 추려두는 곳.
> 이직 시점에 여기서 골라 `문제 → 해결 → 정량 결과 → 배운 것` 형태의 독립 케이스 스터디로 깊게 다시 쓴다.
> 회고가 발행돼도 이 파일은 `_` 접두사라 빌드에서 제외됨.

## 작성 기준 (이게 있어야 "증거")
- [ ] **문제**가 구체적이다 (누가, 무엇이 불편했나)
- [ ] **해결**에 내 의사결정이 들어갔다
- [ ] **정량 결과**가 있다 (수치, 비교 기간)
- [ ] **배운 것**이 다음 행동을 바꿨다
