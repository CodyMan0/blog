RootStoreProvider는 Next.js App Router의 SSR/CSR 구조에 적합하게 zustand store를 Context를 통해 전역 공급하는 구조야.
useRootStore는 selector 패턴과 hydration-safe 로직을 추가한 커스텀 훅이고, 상태 접근의 진입점 역할을 해.
