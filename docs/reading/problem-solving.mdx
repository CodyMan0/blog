48p. 문제를 적확하게 설정하면 올바른 해결책이 나온다.

53p. 문제 해결의 흐름은 문제를 정리하고 구조화하여 진짜 문제를 명확히 한다 -> 가설을 세워 해결책 분석한다 -> 가설 검증한다. -> 실행한다.

56. 문제해결을 위해서 늘 MECE를 의식한다.

57. MECE의 가장 큰 목적은 현상을 제대로 파악하는데 있다.

106p. 좋은 기준을 세우면 세울수록 블루 오션, 즉 경쟁 상대가 없고 고객이 원하는 영역을 찾기 쉬워짐.
